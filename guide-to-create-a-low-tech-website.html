Guide to create a low-tech website

So 10 Mai 2020
By Me
Category: misc

Let's create this website!

The basic website

  1. Download the archive and unzip it. You can changes the paths to the folders images and dithers in the 'dither.py' file.
  2. Install Pelican sudo pip3 install pelican hitherdither
  3. When you're ready you can run sudo sh gen.sh and check your website at https://localhost:8000

Write content

  1. Create content by creating a MD file with the same model as first.md in the folder content.
  2. To add an image you can write ![]({static}/dithers/yourImage.png) and place your image into the folder 'images'

Customize the theme

  1. Press F12 then go into Sources > static/css > style.css
  2. You can now edit the CSS file Do not reload or all your changes will be lost. Once you're happy of your changes, you can copy and paste the content into the file theme/static/css/style.css
  3. If you want to add new elements on all pages, you can edit the file 'base.html' of the folder theme/template

Publishing

  1. You now need to run your server to provide the files. You can use a solar powered server like on http://solar.lowtechmagazine.com
  2. Add the commands to send a file to the provider at the end of 'gen.sh'
  3. Run sudo sh gen.sh
